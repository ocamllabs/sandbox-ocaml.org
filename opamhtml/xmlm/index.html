Xmlm
Streaming XML IO.
A well-formed sequence of TODO reference : #TYPEsignal represents an TODO reference : http://www.w3.org/TR/REC-xml document tree traversal in depth first order (this has nothing to do with XML well-formedness). Input pulls a well-formed sequence of signals from a data source and output pushes a well-formed sequence of signals to a data destination. Functions are provided to easily transform sequences of signals to/from arborescent data structures.
Consult the TODO reference : #io and TODO reference : #ex of use.
Release 1.1.1 - Daniel Bünzli <daniel.buenzli at erratique.ch>

References

  • Tim Bray et al. TODO reference : http://www.w3.org/TR/xml-names11, 2006.
  • Tim Bray. TODO reference : http://www.xml.com/axml/axml.html, 1998.
